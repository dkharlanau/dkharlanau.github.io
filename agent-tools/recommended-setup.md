<section class="section note-detail"><p class="eyebrow">Working setup</p><h1>Recommended agent setup for SAP consultants</h1><p class="lead">Keep knowledge, repeatable procedures, product facts, and system access as separate layers.</p><div class="note-body"><ol><li>Use the reviewed Atlas indexes for diagnostic knowledge.</li><li>Install only the required local agent skills for repeatable procedures.</li><li>Use official SAP documentation for product-specific facts.</li><li>Add MCP tools only after access and authorization are confirmed.</li><li>Require explicit human approval before a write-capable tool is enabled or called.</li></ol><p>This order reduces accidental authority. It also produces an incident brief that can be reviewed even when no live-system tool is available.</p></div></section>
