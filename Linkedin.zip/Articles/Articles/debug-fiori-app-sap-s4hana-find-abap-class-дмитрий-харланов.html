Debug a Fiori app in SAP S4/HANA to find ABAP class?

Created on 2022-01-17 09:45

Published on 2022-01-17 10:54

This is a step-by-step guide to get a trick of exploring ABAP class hidden in a Fiori app. You can not by the command /h jump into the debugger session when you work with Fiori apps. Therefore, you should use the Browser's developer tools to see the POST details to the backend system.

Steps are simple to follow:

1) Run /UI2/FLP - SAP Fiori Launchpad;

2) Select Fiori app ( example with the app 'Manage Purchase orders'):

3) Press the button F12 to launch Developer Tools;

4) See the details in the Network, it will contain a string with OData Service external name:

5) Call the transaction /IWFND/MAINT_SERVICE - Activate and Maintain Services to find ODATA service, select the service and press the button 'SAP Gateway Client':

6) Find out the Data class by pressing the Button 'Service implementation':

7) Use the class's method to put the external breakpoint at the command line.

At the next event in the app, if it calls the method of the class a debugger session will be opened.

More details about SAP Fiori and SAPUI5 Debugging the User Interface you can find in e-Bit book.
