# Dataset Discovery

Use this skill when you need to find the right public dataset, byte collection, or evidence page on the site.

Preferred sources:

- Dataset manifest: https://dkharlanau.github.io/datasets/manifest.json
- Dataset schema: https://dkharlanau.github.io/datasets/schema.json
- Dataset library page: https://dkharlanau.github.io/datasets/
- AI catalog: https://dkharlanau.github.io/ai/catalog.json

Use this skill for:

- locating AMS, DAMA, TRIZ, AI business signal, or agentic byte datasets
- finding canonical dataset landing pages
- routing from a topic to a machine-readable dataset artifact

Avoid using it for:

- protected or private data access
- pretending the site exposes a live API
- inferring unpublished records
