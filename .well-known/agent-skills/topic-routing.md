# Topic Routing

Use this skill when a question is narrow and should be routed to the best topic-specific source instead of the general profile.

Preferred sources:

- AI routing hub: https://dkharlanau.github.io/ai/
- Discovery map: https://dkharlanau.github.io/ai/discovery-map.json
- LLM manifest: https://dkharlanau.github.io/llms.txt

Use this skill for:

- intent-based routing across SAP AMS, operational continuity, integration reliability, MDG, and practical AI topics
- selecting the best canonical page before retrieval
- narrowing broad profile questions into the right evidence path

Avoid using it for:

- tasks that require executable tools or transactions
- assuming API, OAuth, MCP, or WebMCP capabilities that the site does not publish
- treating HTML discovery links as equivalent to HTTP response headers
